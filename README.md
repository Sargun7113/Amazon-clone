# Amazon Clone

## Description
A responsive e-commerce web application replicating core Amazon features. Built using MERN stack with clean UI and scalable architecture.

## Technologies Used
- Frontend: HTML, CSS, JavaScript, React.js
- Backend: Node.js, Express.js
- Database: MongoDB
- Version Control: Git & GitHub

## Features
- User authentication and profile management
- Product listing with search and filters
- Shopping cart functionality
- Clean and responsive UI

## Setup Instructions
1. Clone the repository: `git clone https://github.com/Sargun7113/Amazon-clone.git`
2. Install dependencies: `npm install`
3. Start the server: `npm start`
4. Open `http://localhost:3000` in your browser
